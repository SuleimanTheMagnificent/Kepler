Kepler

A Marginally Inefficient IDE.

About

Kepler is an IDE programmed entirely in the C programming language.

It will utilize the full capabilities of GTK+ to create an idealized programming environment.

Kepler is licensed under GPL and always will be open-source.

Planned features include:

  • Support and syntax-highlating for as many languages as possible
  • Colaboration between users
  • Custom deployable server for colaboration
  • Voice calling and messaging between users
  • Built-in HTML viewer via WebKit
